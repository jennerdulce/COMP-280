# Coding Style

## Naming

### Define
- All names created with #define are to be entirely uppercase
- Use underscores
```c
#define BUFFER_SIZE 20
```

### Variables
- Lower Camel Case
```c
int currentLength = 0;
```
- Snake Case
```c
int current_length = 0;
```

### Typedef
- Names are in Upper Camel Case
```c
typedef unsigned char Byte;
```
